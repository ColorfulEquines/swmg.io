# Full Commands List

Here you can find a list of all the commands for the bot

# Dice Rolling

CommandDescription
/roll [expression] [num_rolls] [secret] [title]Roll using a expression. Optionally provide how many times to run the provided expression, keep the message hidden from public, or provide a title on the roll.

# Initiative

CommandDescription
/initiative create [title]Creates a new initiative session with the provided title.
/initiative end [title]End a initiative session with the provided title.
/initiative join [initiative_name] [character_name]Joins a initiative session with the provided character.
/initiative remove [initiative_name] [character_name]Remove a character from the provided initiative session.
/initiative edit [initiative_name] [character_name] [roll]Modify a character's roll in the provided initiative session.

# Character

CommandDescription
/character create [name]Creates a new character.
/character delete [name]Deletes a character with the provided name.
/character edit [name]Edits a character the provided name.

# Testing

CommandDescription
/pingReplies with 'pong'
/pongReplies with 'ping

# Information

CommandDescription
/helpView information about how to use the bot!
/infoDisplays info about the bot.
